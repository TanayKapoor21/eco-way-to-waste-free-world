# **App Name**: EWWW: Eco Way to a Waste-Free World

## Core Features:

- AI Waste Classifier: Uses the Google Vision API and OpenAI GPT-4 as a tool to classify waste from uploaded images and suggests disposal methods.
- Recycling Center Locator: Uses the Google Maps API to display nearby recycling centers sorted by waste type.
- AI Chatbot: OpenAI GPT-4 powered chatbot assists users with waste disposal queries, offering methods based on the item described by the user.
- Carbon Footprint Tracker: Uses the Climatiq API to track the userâ€™s carbon footprint based on their waste recycling and disposal actions.
- Community Forum: A platform for users to share tips, ideas, and local initiatives related to waste reduction and recycling.

## Style Guidelines:

- Primary color: Forest green (#34A049) to reflect nature and environmental focus.
- Background color: Very light green (#F0F8F0) to give a clean, fresh feel.
- Accent color: Yellow-green (#96D95D) to highlight interactive elements.
- Secondary Accent Color: Light blue (#ADD8E6) for a calming, informative feel.
- Neutral color: Light gray (#D3D3D3) for borders and subtle UI elements.
- Body and headline font: 'PT Sans', a humanist sans-serif for a blend of modernity and warmth.
- Use clear, minimalist icons that represent different types of waste and recycling actions.
- Simple, card-based layout with clear sections for waste classification, center locations, and user profile.
- Smooth transitions and subtle animations to enhance user engagement and provide feedback.
